# Standalone AI Debugger Library

Instantly turn on an AI-powered visual debugger on any web application. Features include direct inspect mode (`Ctrl + Shift + D`), dynamic auto-reproduction steps via pointer simulation, real-time error/network tracing, and instant automated cause/fixing analysis report generated directly using Gemini LLM.

## Installation

```bash
npm install ai-debugger
```

## Usage

Simply import the package into your application entrypoint:

```typescript
import 'ai-debugger';
```

Press **`Ctrl + Shift + D`** (or `Cmd + Shift + D` on macOS) to activate inspect mode, click any element, configure your Gemini API Key in the settings (⚙️), and explain your issue to reproduce and analyze.
